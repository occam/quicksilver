Getting Started
Invoke Quicksilver using the HotKey you defined. This will bring up a window where you can use to search for items and build commands.

Finding an item
To search, simply type part of an item's name. Quicksilver will match both the beginning of a name as well as abbreviations. If Quicksilver finds multiple files that match, it will show a list of alternatives. Select the correct item using the arrow keys and you will be able to use the abbreviation in the future.

Choosing an action (optional)
Quicksilver chooses a default action to perform on the item you have selected. If you'd like to explore other options, hit the tab key to move to the action search field. The down arrow will reveal some of the many things you can do.

Performing the command
Once you have selected an item, hit return to perform it.
